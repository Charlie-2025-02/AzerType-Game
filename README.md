# AzerType-Game ⌨️🎮

AzerType-Game est une application web interactive développée en JavaScript permettant aux utilisateurs d'améliorer leur vitesse de frappe au clavier à travers des défis de mots et de phrases.

## 🎯 Objectif du projet

Le joueur doit reproduire correctement les mots ou phrases proposés par l'application afin d'obtenir le meilleur score possible.

## 🚀 Fonctionnalités

- Choix du mode :
  - Mots
  - Phrases

- Génération dynamique des propositions
- Vérification des réponses utilisateur
- Système de score
- Interaction avec le DOM
- Interface responsive

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript

## 📚 Compétences acquises

- Manipulation du DOM
- Gestion des événements JavaScript
- Utilisation des tableaux
- Fonctions JavaScript
- Conditions et boucles
- Gestion des interactions utilisateur
